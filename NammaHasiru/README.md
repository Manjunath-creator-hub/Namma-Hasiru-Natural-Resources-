# Namma Hasiru 🌱

Android Plantation Tracking App built using:
- Kotlin
- Room Database
- WorkManager

## Features
- Add plant species
- Save tree plantation data
- Persistent local storage
- Reminder worker after 90 days
- Green themed UI

## Run Instructions
1. Open project in Android Studio.
2. Sync Gradle.
3. Run on emulator/device.
