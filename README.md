# Namma-Hasiru 🌱

## AI-Powered Plantation Tracking & Sustainability Monitoring App

Namma-Hasiru is an AI-powered Android application developed to monitor and improve the success of tree plantation and seed ball initiatives. The app helps users geo-tag planted saplings, upload plantation details, track survival status, and receive reminders for growth updates.

The project transforms one-time plantation events into long-term sustainability missions by maintaining survival records and generating environmental insights.

---

## 📌 Features

* 🌍 Geo-tagging of planted saplings
* 📸 Plantation photo upload
* 🌱 Tree species tracking
* ⏰ 90-day reminder notifications
* 📊 Survival rate analytics
* 🗺️ Community plantation map
* 🤖 AI-based tree recommendation system
* 💾 Local data storage using Room Database

## 📱 Project Workflow

1. User plants a tree or seed ball
2. User uploads photo and geo-location
3. Plantation data gets stored locally/cloud
4. App reminds user after 90 days
5. User updates plant survival status
6. App calculates survival analytics and insights

---

## 🎯 Problem Statement

Many plantation drives fail to track whether planted saplings survive after planting. Due to the absence of follow-up monitoring and survival data, the long-term environmental impact of plantation campaigns remains unknown.

Namma-Hasiru solves this problem by providing a digital plantation tracking system that enables users to monitor plant growth, maintain survival records, and improve ecological sustainability.
